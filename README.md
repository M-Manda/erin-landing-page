# Erin Systems Landing Page

A modern, responsive landing page for **Erin Systems LLC**, built using HTML, CSS, and JavaScript. The project showcases a corporate software company website with a clean UI, animated elements, and responsive navigation.

---

## Features

- Responsive landing page layout (desktop + mobile navigation)
- Dark mode toggle (JavaScript feature)
- Animated hero section with fade-in effects
- Company partners logo carousel section
- Clean, modern UI design
- Footer with contact information and social media links

---

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts:
  - Inter
  - DM Sans
  - Source Sans 3
